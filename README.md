# DIO_Excel_Dashboard

O arquivo contém um Dashboard desenvolvido a partir de dados de planos da XBOX.

## Dados

A base de dados contém planos de acesso a jogos, sendo eles: Core, Standard e Ultimate. 
A eles pode ser adicionado outros dois serviços: EA Play Season Pass e Minecraft Season Pass, que dão acessos adicionais.
Além disso, as inscrições podem ser: mensais, anuais e quadrimestrais.

## Descrição do Dashboard
Visa entregar de forma visual as seguintes informações:
- Aumento da adesão de planos durante o ano (o gráfico varia conforme o filtro de *Plan*)
- Total de assinaturas do EA Play Season Pass (em reais) (variam com o filtro de *Period*)
- Total de assinaturas do Minecraft Season Pass (em reais)

### Filtros
- Plan:
  Varia o gráfico de SUBSCRIPTIONS GROWTH conforme os planos, mostrando as novas adesões de todos os planos, ou de forma individual.
- Period:
  Varia os gráficos de TOTAL EA PLAY/MINECRAFT SEASON PLAY, mostranto a receita adquirida através das modalidades de periodo dos planos.
